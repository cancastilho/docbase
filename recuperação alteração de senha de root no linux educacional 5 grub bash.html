Restaurando ou trocando a senha de root no LE 5

1. Ligue o computador. No menu do GRUB selecione a linha "Linux Educacional 5.0, com Linux 3.5.0-30-generic".

2. Aperte a tecla 'e' para editar a linha de inicilização do sistema.

3. Modifique a linha: 

linux /boot/vmlinuz-3.5.0-30-generic root=UIID=NUMERO_GIGANTE ro quiet splash $vt_handoff

para

linux /boot/vmlinuz-3.5.0-30-generic root/dev/sda1 rw quiet splash init=/bin/bash

4. Aperte F10 ou CTRL+X.

Abrirá um terminal no modo root. Neste terminal digite o comando:

passwd

5. Insira a nova senha duas vezes e pronto. A senha do root foi alterada.

Caso queira mudar a senha de algum outro usuário, ex: professor, digite:

passwd professor

... e insira duas vezes a nova senha para o usuário professor.

6. Reinicie o computador.

7. Inicie o computador normalmente e acesse o usuário com a nova senha definida.

Observação: o teclado USB pode não funcionar no shell aberto pelo grub, portanto, use um teclado PS/2.
