Para instalar a ferramenta de cobertura de código do python coverage.py use o comando:

	python -m pip install coverage

Coverage.py permite rodar o script e gerar um relatório de quais partes foram executadas e quais não foram. Execute o seu script com o comando:

	python coverage run seuScript.py

Depois solicite a geração do relatório de cobertura de código com o comando:

	python coverage html

O diretório htmlcov será criado. Entre no diretório e abra o index.html. Lá será possível ver os trechos de código que foram executados e a porcentagem geral de quantas linhas foram executadas no seuTest.py.
