Como fazer encaminhamento de porta no Windows

Para adicionar o encaminhamento:

netsh interface portproxy add v4tov4 listenport=8081 listenaddress=172.16.0.248 connectport=6080 connectaddress=172.16.0.248

Para remover o encaminhamento:

netsh interface portproxy delete v4tov4 listenport=8081 listenaddress=172.16.0.248

Para mostrar os redirecionamentos feitos:

netsh interface portproxy show all

É possível que o redirecionamento trave, sem explicação até o momento. Neste caso, remova o redirecionamento e adicione-o novamente.
