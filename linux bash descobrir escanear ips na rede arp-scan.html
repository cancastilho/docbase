Como vascuhar os ips em uma rede?

No linux instale o arp-scan:

sudo apt-get install arp-scan

Após estar conectado na rede execute:

sudo arp-scan --interface wlan0 192.168.1.0/24    ou  
sudo arp-scan --interface wlan0 --localnet

O comando irá retornar os ips ativos.

Note que --interface indica qual interface utilizar para escanear os ips.
